JLtools CVS Access

  1. If you do not have a CVS account set up for you on ensemble01.cs.cornell.edu:
    1. If you don't have a ssh public key, then run ssh-keygen. If you do not want your key to be protected by a passfrase, then press Enter when asked for one. Make sure you are using the ssh-keygen from ssh version 1.
    2. Send your public key (~/.ssh/identity.pub) and your desired login to Aleksey Nogin (nogin@cs.cornell.edu)
  2. Put the following into your ~/.ssh/config file (if you do not have ~/.ssh/config file, then create one): 
    Host ensemble01.cs.cornell.edu
Port 80
FallBackToRsh no
ForwardX11 no
User <your login on ensemble01>
  3. Set the environment variable CVS_RSH to ssh.

    On Unix, add this to your ~/.profile or ~/.cshrc file. On NT, make it part of your environment (Control Panel/System/Environment)

  4. Make sure you are using cvs version >=1.9 (cvs -v prints version). You may need to download it from somewhere but it should be installed already on your machine.
  5. To create a new working copy of the source repository, run this: 
    cvs -d :ext:ensemble01.cs.cornell.edu:/cvsroot checkout jltools
