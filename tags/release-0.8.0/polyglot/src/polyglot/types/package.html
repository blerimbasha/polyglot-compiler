Contains the typesystem and all related classes. This includes the definitions of {@link jltools.types.Type Type} as well as all the concrete classes for it ( for classes and primitives ). It includes the {@link jltools.types.TypeSystem TypeSystem} interface, as well as an implementation {@link jltools.types.StandardTypeSystem Standard Type System} that corresponds to java 1.2 behavior. @see jltools.types.TypeSystem @see jltools.types.Type @see jltools.types.StandardTypeSystem
