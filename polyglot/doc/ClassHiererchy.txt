This document will attempt to describe the class hierarchy of the JLTools

KEY
===
Packages are written, abbreviated, in [brackets.]

Interfaces are written in (parenthesis).

<<*>> = Not yet implemented.

<V>   = Has an entry in NodeVisitor

(A)   = Abstract
(I)   = Interface
(S)   = Contains static members only.
