Contains the AST Nodes and interfaces for manipulating the nodes.

The nodes all extend the interface {@link jltools.ast.Node Node}. Each node has accessor methods and methods to implement passes over the node. The traversal interface is defined in {@link jltools.ast.NodeVisitor NodeVisitor}. Nodes are constructed using a {@link jltools.ast.NodeFactory NodeFactory}. @see jltools.ast.Node @see jltools.ast.NodeVisitor @see jltools.ast.NodeFactory
